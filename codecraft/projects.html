Family Support Center

This is a project I created for the Family Support Center in my town. They wanted quite a bit of custimization for this site. I added many dynamic and custimizable options for them. MAD Freight

This was such a fun project. This client had a brilliant idea for shipping: to have an app and if people were travelling to and from an area, they list it on the app and other people can use them for shipping, especially useful for farmers needing to rent equipment, etc. It was such a fun idea and it stands for Make A Difference Freight. They had the app, we just needed to market it. This was my first website where we made a video for the home page. I love how it turned out. We made instruction pages for using the app. We made advertisement pages. It's such a great website with some fun css animations on the home page and elsewhere!

This site is built in wordpress with elementor and Custom Fields. All artword was created or found by me. I also used lots of CSS animations to make this website really fun and interesting.
